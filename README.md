## 📚 Overview

**CSCLM (Cloud Computing Learning Module)** is an AI-powered chatbot designed specifically for:
- B.E. Computer Science Students
- B.Tech Information Technology Students
- IT Job Seekers

This application helps students learn Cloud Computing concepts through interactive conversations with the Qwen AI model via OpenRouter API.

## 🌟 Features

- 💬 Interactive chatbot powered by Qwen AI (72B Instruct)
- 💡 10 Random suggested prompts on Cloud Computing topics
- 🎨 Custom UI with royal blue theme
- 📝 Multi-line query input and response display
- 🔄 Easy reset and refresh functionality
- 📚 Covers 30+ Cloud Computing topics from fundamentals to advanced
